QUE ES UN ALGORITMO
Conjunto de instrucciones que es necesario ejecutar ordenadamente para la obtecion de un calculo, la sesolucion de un problema
o la realización de una secuencia ordenada de pasos, exente de ambigüedad, que permite la resolución de un problema determinado.

TIPOS DE ALGORITMOS
Cuantitavios
cualitativos

CARACTERISTICAS
ENTRADA: Generar los datos o valores de inicio.
DEFINIDO: Si se sigue el algoritmo dos veces o mas, se obtiene el mismo resultado.
PRECISO: El algoritmo debe ser puntual y claro al mometno de indicar la realización de cada paso.
FINITO; Se refiere que todo algoritmo tinen un inicio y un fin.
SALIDA; al terminar el algoritmo, la salidad de datos tiene relación con los datos de entrada.
